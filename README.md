

# PrimeTrade.ai Assignment

## Objective

Analyze the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance.

## Datasets

1. Historical Trader Data

2. Bitcoin Fear & Greed Index

## Methodology

Data cleaning

Sentiment mapping

Exploratory Data Analysis

Profitability analysis

## Key Findings

Greed sentiment generated the highest average PnL.

Fear sentiment had the highest trading activity.

Extreme Greed showed the highest win rate.
